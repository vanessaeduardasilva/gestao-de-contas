# Gestor de Contas com Alerta

Projeto em Python para controle de contas a pagar, com alertas automáticos de vencimento via Google Agenda.

## Tecnologias
- Python 
- SQLite
- Google Calendar API

## Funcionalidades
- Cadastro de usuários
- Cadastro de contas
- Controle de vencimentos
- Alertas automáticos

## Como executar
```bash
python main.py
